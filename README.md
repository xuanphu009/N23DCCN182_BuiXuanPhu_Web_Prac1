## Mô tả
Bài thực hành 1 - Xây dựng trang danh sách và chi tiết sản phẩm 
sử dụng Next.js App Router, Tailwind CSS và FakeStore API.

## Tính năng
- Hiển thị danh sách sản phẩm từ FakeStore API
- Responsive: Mobile → Tablet → Desktop
- Trang chi tiết sản phẩm theo dynamic route `/product/[id]`
- Rating hiển thị dạng sao

## Công nghệ sử dụng
- Next.js 14 (App Router)
- Tailwind CSS
- FakeStore API

## Hướng dẫn chạy

npm install
npm run dev

Truy cập: http://localhost:3000
